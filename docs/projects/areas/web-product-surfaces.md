# Web Product Surfaces

Owns the Loom web experience inside `app/`, with shared UI from `components/`.

Primary surfaces:

- Home / Observation Deck
- Desk
- Sources / Library
- Reading source pages
- Panels and Patterns
- Weaves
- Coworks
- Uploads and ingestion entry points

Key folders:

- `app/`
- `components/`
- `app/knowledge/`
- `app/sources/`
- `app/wiki/`
- `app/panel/`
- `app/patterns/`
- `app/weaves/`
- `app/desk/`
- `app/coworks/`

Design rule:

The foreground object must be the source, panel, or relation, not page chrome.
